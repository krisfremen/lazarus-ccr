* LongTimer is a descendant of TIdleTimer
* It has extra properties for long OnTimer intervals
** Every Day at a set Time
** Every Week on a set Day at a set Time
** Every Month  on a set Day at a set Time

* Useful for systray apps
* Compiles with Laz 1.6 ->  FPC 2.62 ->

For more info see: http://wiki.freepascal.org/LongTimer
